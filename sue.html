Setting Up Environment

Hi, In this tutorial I'm going to teach you everything you need to get started in programming in Java.

So, the first thing is to download an Java-N-IDE (Integrated Development Environment) this is a kind of Comiler for android to run Java program in mobile.

And for PC users I give you a video tutorial to download and install BlueJ.

But for now, we stick with our JavaNIDE application in android. Here is a link :- Java-N-IDE to download premium Java N IDE version which is costly on playstore but through this link you can easily download this paid application.

After install the application, you just simply need to open an IDE. It will open an interface like this :-

In project name -> Name your project as you want.
Example:- Tutorials.

Package name -> Use com. always as a prefix and anything after it.
Example:- com.anwar

Main Class name :- Anything excluding space.
Example:-helloWorld.
